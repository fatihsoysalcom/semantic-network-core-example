# Semantic Network Core Example

This example demonstrates a simplified semantic network, illustrating a conceptual approach to language-independent meaning. It defines concepts and their relationships (e.g., "IS_A", "HAS_PROPERTY") to derive properties and check facts, moving beyond statistical word associations. The core idea is to represent meaning abstractly, separate from specific linguistic forms.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`
